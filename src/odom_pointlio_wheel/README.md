# odom_pointlio_wheel

## 说明
odom_pointlio_wheel 是一个 ROS2 包，提供了一个融合点云里程计和轮式里程计的里程计融合节点。该节点订阅点云里程计和轮式里程计的话题，使用一个简单的加权平均方法融合两者的里程计信息，并发布融合后的里程计结果。

## 依赖
- ROS2 Foxy 或更高版本

## 参数解析

### 1. 话题 ──────────────────────────────────
  1. Point-LIO 输出话题: `lio_topic`     
  2. 轮式里程计话题 :  `wheel_topic`    

### 2. 坐标系 ────────────────────────────────
  1. 里程计父坐标系:`frame_id`           
  2. 里程计子坐标系:`child_frame"`     

### 3. 过程噪声（越大 = 越信任观测）─────────
  1. 位置过程噪声: `q_pos` [m²/s]   
  2. 速度过程噪声: `q_vel` [m²/s³]  
  3. 姿态过程噪声: `q_att`[rad²/s]  

### 4. LIO 观测噪声 ──────────────────────────
  **值越小 = 越信任 LIO**
  1. 位置噪声: `r_lio_pos`[m²]
  2. 速度噪声: `r_lio_vel` [(m/s)²] 
  3. 姿态噪声: `r_lio_att`[rad²] 

### 5. 轮式观测噪声 ──────────────────────────
  #### vy 设较大：轮式侧向速度不可靠
  1. 前向速度噪声: `r_wheel_vx`[(m/s)²]  
  2. 侧向速度噪声: `r_wheel_vy`[(m/s)²]
  3. (预留，暂未使用):`r_wheel_yaw` [rad²]  

### 6. 轮式外参（轮式坐标系 → LIO body 系）──
  1. x 平移 :`wheel_offset_x` [m] 
  2. y 平移 :`wheel_offset_y` [m]
  3. yaw 旋转: `wheel_offset_yaw`[rad]
